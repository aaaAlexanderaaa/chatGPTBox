# v3.2.0

- Added cached ChatGPT conversation endpoints to the local API Server Bridge, including create, inspect, follow-up, and refresh flows.
- Improved ChatGPT Web bridge reliability on Brave and macOS with safer proxy routing, connection recovery, and buffered websocket handling.
- Redesigned the settings UI and quick settings flow, with better separation between General, Features, Agents, Modules, and Advanced options.
- Updated ChatGPT Web model handling to use the newer dynamic model list and preserve upstream slugs in conversation requests.
- Expanded the API server docs and draft workflow examples for conversation inspection and follow-up automation.
